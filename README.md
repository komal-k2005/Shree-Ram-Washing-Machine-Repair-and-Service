# Shri Ram Enterprises - Appliance Repair Services

Welcome to the **Shri Ram Enterprises** website project! This is a fully responsive, modern web application designed for a premium home appliance repair and diagnostic servicing business located in Pune, Maharashtra.

## 🚀 Features

- **Premium Modern Design**: Built with a luxurious navy blue and gold aesthetic inspired by professional service standards.
- **Responsive Layout**: Completely mobile-friendly, ensuring a seamless user experience across all devices (desktops, tablets, and smartphones).
- **Service Pages**: Dedicated detailed pages for their core services:
  - Washing Machine Repair
  - Refrigerator Repair
  - Microwave Oven Repair
- **Interactive Booking Form**: A fully functional online scheduling form connected to FormSubmit. It sends booking requests directly to `shriramenterprises084@gmail.com` without needing a backend server.
- **WhatsApp Integration**: A floating WhatsApp button on all pages allows customers to instantly message the business for fast support.
- **Micro-animations**: Smooth scroll animations powered by AOS (Animate On Scroll) for a dynamic, engaging user experience.

## 💻 Technologies Used

- **HTML5**: Semantic markup and structure.
- **Vanilla CSS3**: Custom styles, responsive Flexbox/Grid layouts, and CSS variables.
- **Vanilla JavaScript (ES6)**: Frontend validation, dynamic interactions, and interactive form handling.
- **FormSubmit API**: Handles email forwarding for the booking form.
- **AOS Library**: For scroll animations.
- **FontAwesome**: For scalable vector icons.

## 🛠️ How to Run Locally

Since this is a static website, no complex server setup or build step is required!

1. Clone or download the repository to your local machine.
2. Open the project folder (`Shree-Ram-Washing-Machine-Repair-and-Service`).
3. Simply double-click on `index.html` to open it in your default web browser. 

*(Note: For the booking form to successfully send emails to FormSubmit without CORS issues during development, you may want to serve it through a simple local server, such as the VS Code "Live Server" extension, or Python's `python -m http.server`)*
